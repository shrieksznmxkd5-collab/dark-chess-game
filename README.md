# 卧底象棋

一个无后端、无数据库、无账号系统的中国象棋隐藏身份网页小游戏。玩法基于标准中国象棋棋盘：除帅、将和四枚士外，其余 26 枚棋子随机隐藏在标准开局棋位，暗棋完成首次合法行动后翻面并按真实身份继续游戏。

## 本地启动

项目使用原生 HTML、CSS 和 JavaScript ES Modules。不要直接双击 `index.html` 用 `file://` 打开，否则浏览器可能会拦截模块导入，只显示空棋盘外壳。

建议用 VS Code 的 Live Server 插件打开，或在项目根目录启动本地静态服务器：

```bash
python -m http.server 5173
```

然后访问：

```text
http://localhost:5173
```

规则测试页：

```text
http://localhost:5173/tests/rules-test.html
```

也可以在安装了 Node.js 的环境下运行规则测试：

```bash
npm test
```

## 已实现

- 标准 9×10 中国象棋棋盘、楚河汉界和九宫。
- 首页选择人机模式或本地双人模式，进入后才显示棋盘。
- 棋子和可点击落点使用同一套 `row / col` 交叉点坐标映射，响应式缩放后仍对齐棋盘交叉点。
- 6 枚明棋固定开局，26 枚暗棋只在标准开局暗棋棋位中随机洗牌。
- 暗棋同时保存真实身份和临时身份。
- 翻面前按 `initialRole` 和 `initialCamp` 行动，首次行动后揭示真实身份。
- 标准车、马、炮、兵卒、士、将帅、象相规则。
- 马腿、象眼、炮架、九宫、过河、将帅照面和将帅安全检查。
- 延迟威胁状态机：限制方回合与受保护方调整回合。
- 吃掉将或帅才判胜。
- 无合法着法自动跳过，不直接判负。
- 水墨国风界面：首页模式选择、棋盘页状态栏、中央棋盘、底部辅助区。
- 双人模式 / 人机模式切换。人机模式下玩家执红，AI 执黑。
- 简单规则型 AI：从所有合法着法中评分选择，优先吃将帅、吃高价值棋子和处理己方危险。
- 基础移动反馈、合法落点、音效开关和震动开关。

## 本地检查建议

1. 打开首页，确认只显示《卧底象棋》、模式选择和设置。
2. 点击“人机模式”或“双人模式”，进入棋盘页后确认棋子中心正好落在棋盘交叉点上。
3. 缩放浏览器宽度，确认棋子和交叉点仍保持对齐。
4. 点击“人机模式”，确认红方由玩家操作，黑方自动思考并落子。
5. 打开规则测试页，确认规则测试通过。

## 部署

这是纯静态项目，可直接部署到 GitHub Pages 或 Netlify。部署目录选择项目根目录即可，不需要构建命令。
